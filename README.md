# Number Guessing Game

This is a simple Java-based Number Guessing Game.

The program generates a random number between 1 and 100, and the user has to guess the correct number within a limited number of attempts. It provides hints such as "Too High" or "Too Low" after each guess.

## Features
- Random number generation
- Maximum 10 attempts
- Input validation
- Win/Lose messages
